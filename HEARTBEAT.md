# HEARTBEAT.md

## Work Check (every heartbeat)
1. Read WORKLEDGER.md for current tasks
2. Check active work status
3. If blocked, note in NOTES.md
4. If complete, move to next backlog item
5. Update WORKLEDGER.md with progress

## Research Scout (1x daily, morning UTC)
If 24h+ since last scout:
1. Run `./research/scout.sh hunt "agentic AI"`
2. Log interesting findings to research/log.md
3. Add promising items to WORKLEDGER.md backlog
